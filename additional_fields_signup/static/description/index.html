Additional Fields Signup Form

Este módulo agrega número de teléfono, dirección y fecha de nacimiento a la página de registro de autenticación

Setting

Vaya a Ajustes> Configuración general> Usuarios

En Usuarios seleccione Registro gratuito

Permite a tus nuevos usuarios obtener una opción de registro..

Fields

  • Teléfono
  • Celular
  • Dirección
  • Ciudad
  • C. Postal
  • Provincia
  • Pais
  • Fecha de nacimiento
Nota: Este módulo está probado y funciona en Odoo Community Edition pero no está probado en Enterprise Edition.
